# Test: Basic Calculator App

## What it tests
- Single artifact generation
- File creation (HTML, CSS, JS)
- Vite dev server start
- Basic interactivity

## Prompt
```
Build a scientific calculator with a clean dark theme. It should support basic operations (+, -, *, /), plus sin, cos, tan, log, sqrt, and exponents. Include keyboard support and a history panel that shows the last 10 calculations.
```

## Expected behavior
- Creates package.json with vite
- Creates index.html, styles, and JS/TS files
- Starts dev server
- Calculator is functional with all listed operations
